documentation   download   development  

Introduction

rQuilt is a "cherry-picking" XML parser for Ruby, inspired by E4X and jQuery.

While coding a e4x implementation in Ruby is possible (see R4X), the standard e4x syntax is not. So rather then focus on the details of parser implementation, rQuilt adapts a Ruby-esque front-end designed in the spirit of e4x, but which works via previoulsy tried-and-true XML parsers on the back-end. Specifically, rQuilt has REXML and Ruby-LibXML adapters, either of which can drive the easy to use rQuilt interface. This is very useful when in need of a pure Ruby setup using REXML, or need the speed of LibXML.

In recent updates rQuilt has also drawn inspiration from the Hpricot HTML parser. This is a gret library in the same spirit. So rQuilt's API has been made increasingly compatible with it for better interoperability --and to save you brain cells.

Intstallation

Installation is a simple as ever with RubyGems:

        gem install rquery

Manual installation is provided via setup.rb.

Usage

Let's say we have this XML document stored in inventory.xml:

        <inventory title="OmniCorp Store #45x10^3">
          <section name="health">
            <item upc="123456789" id="12">
              <name>Invisibility Cream</name>
              <price>14.50</price>
              <description>Makes you invisible</description>
            </item>
            <item upc="445322344" id="18">
              <name>Levitation Salve</name>
              <price>23.99</price>
              <description>Levitate yourself for up to 3 hours per application</description>
            </item>
          </section>
          <section name="food">
            <item upc="485672034" id="653">
              <name>Blork and Freen Instameal</name>
              <price>4.95</price>
              <description>A tasty meal in a tablet; just add water</description>
            </item>
            <item upc="132957764" id="44">
              <name>Grob winglets</name>
              <price>3.56</price>
              <description>Tender winglets of Grob. Just add water</description>
            </item>
          </section>
        </inventory>

Now we can rquery information from this document, like so:

        root = RQuery::Xml.load('inventory.xml')

        root['title']         #=> "OmniCorp Store #45x10^3"

        root/:section         #=> <section name='health'> ... ,
                                  <section name='food'> ...

        root/:section/:item   #=> <item id='12'  upc='123456789'> ... ,
                                  <item id='18'  upc='445322344'> ... ,
                                  <item id='653' upc='485672034'> ... ,
                                  <item id='44'  upc='132957764'> ...

The selection process allows you to quickly select large portions of document as well as narrow in on particulars.

Have a look at the API documentation for more information.
