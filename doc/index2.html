jQuilt
A jQuery Plugin

jQuilt is a "weaving" template system that operate fully on the front-end, thus off-loading some of the back-end work load while also providing 100% SOC (Seperation of Concern).

Examples


       data = {
         hello: "Hello, World!"
       }

       $.jquilt(data)

jQuilt templates are especially useful when filled via AJAX with JSON data. For this reason a shortcut is provided. 


        $.jquilt_ajax('http://somewhere/data.json')
